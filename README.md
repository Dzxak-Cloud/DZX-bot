## Heroku Buildpack
```bash
heroku/nodejs
https://github.com/jonathanong/heroku-buildpack-ffmpeg-latest
```

## For Termux
```ts
termux-setup-storage
apt update && apt upgrade
pkg install nodejs git ffmpeg libwebp imagemagick
git clone https://github.com/Dzxak-Cloud/DZX-private.git -b V2
cd DZX-private
pkg install yarn
yarn
npm install
npm start
```
